Beshto

 

This is a social network designed for Bangladeshi people. It allows user to communicate with each other, discuss different topic. It also has a Question and Answer section where user can ask question and other user give answer. Groovy and Grails used for this project with Spring security support. Database for this project is mySql. SpringSource tool Suit is used as IDE, Ubuntu used as developing environment, Apache tomcat is the application server. I work as a member of the development team. My main job was work with controller where I had to write code in groovy and also the view where I wrote code in groovy server page.

Ebajar

 

This is one of the product of Springine. This is a java application with spring framework. All the configuration associated with this site has been done by me. E.g. Configuring Spring Framework, Site mesh, Bootstrap, Maven, MySQL, Hibernate, JDBC connection, tomcat 7 server, configuring amazon web service, setting up the domain (from yahoo.com) for proper redirection, managing github. This is a best deal type application. User can find best deal for their desired product, add there product in the wish list or send the wish list to their friends. User also gets coupon based on their choice of products in their wish list.
