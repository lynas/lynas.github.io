Postgresql Setup

Download zip

https://www.enterprisedb.com/download-postgresql-binaries

Setup db

Create data directory inside zip extracted folder

Inside bin directory run following command [oneTime only]

initdb.exe -D ../data -U postgres -W -E UTF8 -A scram-sha-256

Start database

pg_ctl -D ../data -l logfile start

To shutdown Postgres, run

pg_ctl -D ../data stop
